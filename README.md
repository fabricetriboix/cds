Getting started
---------------

    $ scons --help
    $ scons --mkdoc -j4    # Build
    $ ./run_unit_tests.py  # Run the unit tests
    $ ./install.py --help
    $ ./install.py         # Install

Read the doxygen documentation to learn how to use cds.
